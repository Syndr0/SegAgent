# Guideline corpus

Place institution-approved Markdown or text guidelines here. Keep one document
per source and include a title, version/date, jurisdiction and stable source URL
near the top. Headings become retrievable section labels.

This folder intentionally contains no fabricated clinical guidance. The seed
protocol registry is sufficient for software tests, but guideline-grounded
experiments should fail or abstain until real, authorized source documents are
added and retrieval quality is evaluated.
